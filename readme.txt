== Description ==
Embeds youtube list items on your page using shortcodes

== Installation ==
1 - Dowload plugin
2 - rename config.example.php to config.php
3 - add your api key to config.php file
4 - use shortcodes as [cylist youtube_list_id=\"enter youtube list id\"]
